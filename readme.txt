**How does it work?**

Our plugin adds a specially code which adds the option to change variants by changing the image in the gallery.

== Installation ==
1. From the WordPress menu, select 'Plugins' and then 'Add New'.
2. Select 'Send plugin to server'.
3. Click the 'Browse' button and select the .zip file with the plugin.
4. Click the 'Install' button and then activate the plugin.
